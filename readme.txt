Responsive HTML & CSS3 accordion for WordPress.

== Description ==

Accordion's wp is a fully responsive and jQuery WordPress plugin that offering a modern and engaging user experience.


<strong>Plugin Features </strong>

* Fully responsive.
* All Browser support.
* Unlimited accordions.
* Use via short-code.
* 5 Different Themes.
* Custom post enable.
* Custom accordions background color.
* Custom default background color.
* Custom font color and size for accordions header.
* Custom font color and size for accordions content.
* Easy option panel.



== Installation ==

1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. After activate plugin you will see "Accordion" menu at left side on WordPress dashboard and find "Accordion Option settings"<br />
<br /><br />

<strong>How to use short-code?</strong><br />
When accordions options setup done please paste [accordion-themepoints] this shortcode into your theme any where you want to display this accordion.
